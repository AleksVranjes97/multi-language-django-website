Pythagorean Theorem (with Go)

The Pythagorean Theorem is another simple but cool mathematical equation:

There are four functions in a pre-compiled Go file in /calculation/scripts/ that a Foreign Function Interface accesses to print the hypotenuse of four different sized triangles:

A relatively small triangle:

A medium-sized triangle:

Hypotenuse = {{ hypotenuse.mediumHypotenuse }}

A pretty large triangle:

Hypotenuse = {{ hypotenuse.largeHypotenuse }}

A MASSIVE triangle:

Hypotenuse = {{ hypotenuse.extraLargeHypotenuse }}
